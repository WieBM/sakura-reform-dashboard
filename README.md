# さくらリフォーム 経営ダッシュボード

さくらリフォームの社長向け業務管理システムです。受注状況・請求・顧客・社員情報をブラウザ上で一元管理できます。

---

## 起動方法

```bash
cd dashboard
npm start
```

ブラウザで **http://localhost:3000** を開いてください。

> 初回起動時は `HandsOn_資料/` 内の Excel ファイルを自動的に読み込み、データベースを作成します。2回目以降はスキップされます。

---

## 画面構成

左側のサイドバーから各ページに切り替えます。

| ページ | 内容 |
|--------|------|
| 📊 ダッシュボード | KPI・グラフ・最近の受注案件 |
| 🏗️ 案件管理 | 全案件の進捗確認・追加・編集・削除 |
| 💳 請求・入金 | 請求データの管理・入金状況の更新 |
| 👥 顧客台帳 | 顧客情報の管理・追加・編集・削除 |
| 👷 社員管理 | 社員情報と担当案件数（業務量）の確認 |
| 📈 担当者実績 | 担当者別の受注件数・受注金額ランキング |

---

## 各ページの使い方

### 📊 ダッシュボード

経営状況を一目で把握できます。

- **受注総額** ― 「契約済」案件の契約金額の合計
- **見込み金額（パイプライン）** ― 契約前の進行中案件の見込み合計
- **入金済み総額** ― 請求書ステータスが「入金済」の合計
- **案件数** ― 全案件の件数（失注件数も表示）

グラフは3種類あります：
1. **案件ステータス別** ― 各ステータスの案件件数の内訳（ドーナツグラフ）
2. **工事種別 受注・見込み金額** ― 種別ごとの金額比較（棒グラフ）
3. **月別 受注金額** ― 月ごとの契約金額の推移（折れ線グラフ）

---

### 🏗️ 案件管理

全案件をリスト表示します。

**フィルタ**：ページ上部のボタンでステータスを絞り込めます。

| ステータス | 意味 |
|-----------|------|
| 初回訪問済 | 初めて現地を訪問した段階 |
| 現地調査済 | 現場調査が完了した段階 |
| 商談中 | 見積もり前に交渉中の段階 |
| 見積提出済 | 見積書を提出した段階 |
| 契約済 | 正式に契約が成立した段階 |
| 完了 | 工事が完了した案件 |
| 失注 | 受注に至らなかった案件 |

**検索**：案件名・顧客名でリアルタイム検索ができます。

**新規案件を追加**：右上の「＋ 新規案件を追加」ボタンをクリックするとフォームが開きます。

**編集・削除**：各行の「編集」「削除」ボタンから操作できます。

**担当者の変更**：担当者名をクリックすると担当者選択モーダルが開き、社員一覧から選択して変更できます。

---

### 💳 請求・入金管理

請求書の一覧と入金状況を管理します。

- ページ上部に「入金済み合計」「未入金合計」「請求件数」のKPIが表示されます
- フィルタボタンで「入金済」「未入金」に絞り込めます
- 「＋ 新規請求を追加」から新しい請求書を作成できます
- 各行の「編集」で入金日・入金状況を更新できます

**入金状況の種類**：`未入金` / `入金済`

---

### 👥 顧客台帳

顧客情報を一覧・管理します。

- 氏名・住所でリアルタイム検索できます
- 「＋ 新規顧客を追加」から新規顧客を登録できます
- 各行の「編集」から情報を更新できます
- 顧客を削除すると、その顧客が担当していた案件の担当者欄が自動的に空欄になります

---

### 👷 社員管理

社員情報と業務負荷を一覧で確認できます。

各社員カードに以下の情報が表示されます：
- 氏名・部署・役職・資格
- 担当案件数と業務量バー
- 業務レベル（**余裕** / **普通** / **多忙**）

業務レベルの基準：

| レベル | 案件数の目安 |
|--------|-------------|
| 🟢 余裕 | 1件以下 |
| 🟡 普通 | 2〜3件 |
| 🔴 多忙 | 4件以上 |

「＋ 新規社員を追加」から社員を登録できます。社員を削除すると、担当していた案件・顧客の担当者欄が自動的に空欄になります。

---

### 📈 担当者別実績

担当者ごとの受注成績をランキング形式で確認できます。

| 列 | 内容 |
|----|------|
| 担当案件数 | 担当しているすべての案件件数 |
| 受注件数 | 「契約済」「完了」の案件件数 |
| 受注率 | 受注件数 ÷ 担当案件数 |
| 受注金額 | 契約金額の合計 |
| 受注シェア | 全体受注金額に対する割合（進捗バー表示） |

## データについて

### 元データ（Excel）

`HandsOn_資料/` フォルダ内の Excel ファイルが元データです。初回起動時に自動インポートされます。

| ファイル | 内容 |
|----------|------|
| 社員名簿.xlsx | 社員マスタ |
| 案件管理表.xlsx | 案件情報 |
| 顧客管理台帳.xlsx | 顧客情報 |
| 見積明細一覧.xlsx | 見積データ |
| 請求・入金管理表.xlsx | 請求・入金データ |
| 工事サービス標準単価表.xlsx | サービス単価 |

### データベース

`dashboard/sakura.db`（SQLite）にデータが保存されます。ブラウザ上で追加・編集・削除した内容はリアルタイムにデータベースへ反映されます。

## よくある操作

**案件のステータスを「契約済」に更新したい**
→ 案件管理ページ → 対象行の「編集」→ ステータスを「契約済」に変更 → 「保存する」

**請求書の入金が確認できた**
→ 請求・入金ページ → 対象行の「編集」→ 入金状況を「入金済」、入金日を入力 → 「保存する」

**新しいお客様が来た**
→ 顧客台帳ページ → 「＋ 新規顧客を追加」→ 必要情報を入力 → 「保存する」

**案件の担当者を変更したい**
→ 案件管理ページ → 担当者名をクリック → 社員一覧から選択
