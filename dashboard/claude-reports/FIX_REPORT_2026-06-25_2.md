## 修正詳細

### BUG-01 — チャート再生成クラッシュ修正

**変更箇所**: `loadDashboard()` 直前 + 各 `new Chart()` 呼び出し

モジュールスコープにインスタンス変数を追加し、再生成前に `.destroy()` を呼ぶよう変更。

```js
// 追加
let chartStatusInst = null, chartTypeInst = null, chartMonthlyInst = null;

// 変更前
new Chart(document.getElementById('chartStatus'), { ... });

// 変更後
if (chartStatusInst) chartStatusInst.destroy();
chartStatusInst = new Chart(document.getElementById('chartStatus'), { ... });
```

`chartTypeInst`・`chartMonthlyInst` も同様に対応。

### BUG-05 — 社員編集モーダルのキャッシュ使用

**変更箇所**: `openEmployeeModal()` の編集分岐

既存の `allEmployees` キャッシュを優先使用し、空の場合のみフェッチするよう変更。

```js
const fillEmpForm = (emps) => { /* フォームセット処理 */ };

if (allEmployees.length) {
  fillEmpForm(allEmployees);
} else {
  fetch('/api/employees').then(r=>r.json()).then(emps => {
    allEmployees = emps;
    fillEmpForm(emps);
  });
}
```

### BUG-06 + BUG-11 — Stored XSS・HTMLエスケープ対応

**変更箇所**: `esc()` 関数の直後に `html()` ヘルパーを追加。全 `insertAdjacentHTML` テンプレートに適用。

```js
// 追加
function html(str) {
  return (str||'').replace(/&/g,'&amp;').replace(/</g,'&lt;').replace(/>/g,'&gt;').replace(/"/g,'&quot;');
}
```

適用箇所:

### BUG-09 — 全 fetch 呼び出しにエラーハンドリング追加

**変更箇所**: 全 `loadXxx()` 関数 + 全 CRUD submit/delete ハンドラ

```js
// loadXxx() パターン
try {
  allProjects = await fetch('/api/projects').then(r=>r.json());
} catch(err) {
  showToast('案件データの取得に失敗しました', 'error');
  return;
}

// CRUD submit パターン
try {
  const res = await fetch(...);
  const json = await res.json();
  if (!res.ok) { showToast(json.error || 'エラーが発生しました', 'error'); return; }
  // 成功処理...
} catch(err) {
  showToast('通信エラーが発生しました', 'error');
}

// delete パターン
try { res = await fetch(`/api/xxx/${id}`, { method: 'DELETE' }); }
catch(err) { showToast('通信エラーが発生しました', 'error'); return; }
```

対象関数: `loadDashboard`・`loadProjects`・`loadInvoices`・`loadCustomers`・`loadStaff`・`loadEmployees`・`customerForm submit`・`projectForm submit`・`employeeForm submit`・`invoiceForm submit`・`deleteCustomer`・`deleteProject`・`deleteEmployee`・`deleteInvoice`
